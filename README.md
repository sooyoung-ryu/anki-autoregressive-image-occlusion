# Autoregressive Image Occlusion

Reveals image occlusion masks one at a time on each spacebar press, then automatically shows the answer when the last mask is revealed.
Cards with a single mask behave exactly as normal.

### When to use
Works especially well for understanding code snippets.

### Example
For a card with three masks on the same group:

Initially, all masks are hidden.

Pressing Spacebar → Reveals the first mask (on the top)

Pressing Spacebar again → Reveals the second mask

Pressing Spacebar again → Reveals the last mask (on the bottom) and shows the answer.

Masks are revealed top-to-bottom, then left-to-right.

### Installation

**Via AnkiWeb:** Search for "Autoregressive Image Occlusion" in Tools > Add-ons > Get Add-ons.

**Manual:** Download `autoregressive_image_occlusion.ankiaddon` from the releases page and open it with Anki (File > Install Add-on).
